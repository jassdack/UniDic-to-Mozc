# Mozc（Google 日本語入力）ユーザー辞書 対応品詞一覧

実際のMozc辞書ツールにおいて、ドロップダウンメニューで選択可能な**45種類**の品詞とその意味・使われ方の一覧です。
（※ `config/pos_mapping.json` をカスタマイズする際、`result` に指定する文字列は必ず**以下の見出しの文字列（バッククォート内の文字）**と完全に一致させてください。）

## 名詞・固有名詞系

- `品詞なし` : 品詞を特定しない、または文法的な役割を持たない文字列。
- `名詞` : 一般的な物や事柄の名前。活用しません。（例: パソコン, りんご）
- `固有名詞` : 世界に一つしかない特定の事物・ブランドなどの名前。（例: トヨタ, アマゾン）
- `人名` : 歴史上の人物や一般の人名など、人を表す名詞。（例: 田中, アインシュタイン）
- `姓` : 名字（ファミリーネーム）。（例: 佐藤, 鈴木）
- `名` : 下の名前（ファーストネーム）。（例: 太郎, 花子）
- `組織` : 企業、団体、政府機関などの名前。（例: 株式会社〇〇, 国際連合）
- `地名` : 国、都市、山、川などの場所の名前。（例: アメリカ, 北海道, 富士山）
- `数` : 数字や漢数字など、数を表す語。（例: 一, 百, ゼロ）
- `アルファベット` : 英字で構成される文字列。頭文字等。（例: IT, HTTP）

## 活用する名詞系（〜する、〜だ）

- `名詞サ変` : 後ろに「〜する」を付けることで動詞のように機能する名詞。（例: 勉強, 処理, 検索）
- `名詞形動` : 後ろに「〜だ」「〜な」を付けることで修飾語（形状詞・形容動詞）として機能する名詞。（例: 健康, 静か, 綺麗）

## 記号・特殊系

- `記号` : 一般的な記号類。（例: ⭐, ♡, ％）
- `句読点` : 句点や読点。（例: 。 , 、）
- `顔文字` : 複数の記号等で作られた特殊なアスキーアート。（例: (´・ω・`), (^o^)）

## 修飾語・接続語系

- `副詞` : 動詞や形容詞などを修飾する言葉。活用しません。（例: とても, ゆっくり, 非常に）
- `連体詞` : そのままの形で名詞だけを修飾する言葉。（例: この, 大きな, あらゆる）
- `接続詞` : 文と文を繋ぐ言葉。（例: しかし, また, および）
- `感動詞` : 呼びかけや応答、感情などを表す言葉。（例: はい, ああ, おはよう）

## 接辞・助数詞系（他の語にくっつく言葉）

- `接頭語` : 単語の頭にくっついて意味を添える言葉。（例: 【超】かわいい, 【ご】案内）
- `接尾一般` : 単語の後ろにくっついて意味を添える言葉。（例: かわい【さ】, 深【み】）
- `接尾人名` : 人名の後ろに付く敬称など。（例: 田中【さん】, 社長【殿】）
- `接尾地名` : 地名の後ろに付く言葉。（例: 東京【都】, 新宿【区】）
- `助数詞` : 数量を数えるときの単位。数字のあとに付きます。（例: 1【個】, 3【本】, 100【円】）

## 形容詞系（※基本形・終止形 だけで登録します）

- `形容詞` : 「〜い」で終わる言葉。（例: 美しい, 早い, 寒い）

## 特殊な独立機能タグ

- `独立語` : 前後に文脈を持たない独立した言葉。Mozc本来の辞書にはない助詞や助動詞を強制的に登録する際にも便宜上使われます。
- `終助詞` : 文の終わりに付いて話し手の態度等を添える言葉。（例: だ【よ】, 行く【ね】）
- `短縮よみ` : 長い言葉を意図的に短い短い「よみ」で登録する機能。（例: よみ「よろ」→ 語彙「よろしくお願いします」）
- `サジェストのみ` : 変換時には直接出さず、タイプ中のサジェスト（予測変換）の候補として優先表示させるためのタグ。
- `抑制単語` : サジェストして欲しくない（邪魔な）読みを強制的に隠す・変換候補から除外（マイナス）するための特殊タグ。
